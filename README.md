# ofi-node
Test ofi with node, handlebars and angularjs

1.- Clonar repositorio

2.- Descargar las dependencias necesarias para la aplicacion con la instrucción
    npm install
    
3.- Ejecutar la aplicación con
    npm start
    
    ó
    
    node ./bin/www
    

4.- Visualizar el sitio en
    http://localhost:3000
